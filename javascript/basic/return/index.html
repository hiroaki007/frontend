基本的なJavaScriptのデータ型

基本的なJavaScriptのデータ型 一覧 | データ型 | 意味 | 例 | | --- | --- | --- | | Number | 数字 | `10`, `3.14` | | String | 文字列（もじれつ） | `"こんにちは"`, `"ABC"` | | Boolean | true / false（真偽） | `true`, `false` | | Undefined | 未定義 | `undefined` | | Null | 何もない | `null` | | Object | まとめたデータ | `{ name: "Taro" }` | | Array | 配列（はいれつ） | `[1, 2, 3]` |
